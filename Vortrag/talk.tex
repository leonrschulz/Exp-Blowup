\def\nneegg#1{\neg #1}
\begin{frame}{Example}{}
  Conjuncton of 3 xor = x1 xor y1 and x2 xor y2 and x3 xor y3

  $F_3 =
    \overbrace{(x_1 \lor y_1) \land (\nneegg{x_1} \lor \nneegg{y_1})}^{x_1 \mathop{\oplus} y_1} \land
    \overbrace{(x_2 \lor y_2) \land (\nneegg{x_2} \lor \nneegg{y_2})}^{x_2 \mathop{\oplus} y_2} \land
    \overbrace{(x_3 \lor y_3) \land (\nneegg{x_3} \lor \nneegg{y_3})}^{x_3 \mathop{\oplus} y_3}$

  $F_3$ is in CNF and can be transformed in an equivalent formula in DNF:

  \begin{align*}
    & (x_1 \land \nneegg{y_1} \land x_2 \land \nneegg{y_2} \land x_3 \land \nneegg{y_3}) \mathrel{\lor}
      (x_1 \land \nneegg{y_1} \land x_2 \land \nneegg{y_2} \land \nneegg{x_3} \land y_3) \mathrel{\lor} \\
    & (x_1 \land \nneegg{y_1} \land \nneegg{x_2} \land y_2 \land x_3 \land \nneegg{y_3}) \mathrel{\lor}
      (x_1 \land \nneegg{y_1} \land \nneegg{x_2} \land y_2 \land \nneegg{x_3} \land y_3) \mathrel{\lor} \\
    & (\nneegg{x_1} \land y_1 \land x_2 \land \nneegg{y_2} \land x_3 \land \nneegg{y_3}) \mathrel{\lor}
      (\nneegg{x_1} \land y_1 \land x_2 \land \nneegg{y_2} \land \nneegg{x_3} \land y_3) \mathrel{\lor} \\
    & (\nneegg{x_1} \land y_1 \land \nneegg{x_2} \land y_2 \land x_3 \land \nneegg{y_3}) \mathrel{\lor}
      (\nneegg{x_1} \land y_1 \land \nneegg{x_2} \land y_2 \land \nneegg{x_3} \land y_3)
  \end{align*}

  Through the transformation, the formula's size increased from 29 to 119.
\end{frame}

\begin{frame}{Formalization Preliminaries}{}
  Based on \textsf{Propositional\_Proof\_Systems} (Michaelis and Nipkow)
  Plus?

  \begin{itemize}
    \item Fn (Fn): Creates a formula in CNF for some given n (analogously to F3)
    \item size (|.|), sizef: Count the number of logical symbols (sizef does not consider negations)
    \item BigAnd‘ ($\bigwedge$), BigOr‘ ($\bigvee$): Build the conjunction and disjunction of all formulas in a list
    \item dual: Dual of a formula
  \end{itemize}
\end{frame}

\begin{frame}{CNF to DNF}{Low-level}
  \begin{block}{Proposition (Blowup from $F_n$ to $BigOr'$)}
    Let n > 0 be a natural number.
    Let Ts be a list of formulas.
    Let ψ= \/ |Ts| i=0 Ts[i].
    If all formulas in Ts are conjunctive and satisfiable, and if Fn is equivalent to ψ, then sizef ψ ≥ n·2n.
  \end{block}

  \pause

  Proof:
  \begin{enumerate}
    \item<+-> Every T ∈ set Ts contains for all i = {1,...,n} either the variable xi,False or xi,True positively.
    \item<+-> Every T ∈ set Ts has sizef T ≥ n (using step 1).
    \item<+-> For every Boolean list ϵ = [ϵ1,...,ϵn] there exists a T ∈ set Ts that contains the corresponding variables x1,ϵ1 ,..., xn,ϵn positively.
    \item<+-> The number of Boolean lists of length n is 2n ≤ |Ts| (using step 1,3).
    \item<+-> Finally sizef ψ ≥ n·2n (using step 2,4).
  \end{enumerate}
\end{frame}

\begin{frame}{CNF to DNF}{High-level}
  \begin{theorem}[Blowup from CNF to DNF]
    Let n be a natural number.
    There exists a formula ϕ in CNF with size |ϕ| = 10·n+2 such that every equivalent formula ψ in DNF has size |ψ| ≥ n·2n.
  \end{theorem}
  Proof:
  \begin{enumerate}
    \item Case analysis on n gives us n > 0 (case n = 0 can be proven directly).
    \item Instantiate ϕ = Fn and then prove that ϕ is in CNF and |ϕ|=10·n+2.
    \item Assume an equivalent formula ψ in DNF and obtain a list Ts of conjunctive and satisfiable formulas in ψ and show sizef ψ ≥ sizef (\/ |Ts| i=0Ts[i]) and the equivalence of \/ |Ts| i=0Ts[i] and ϕ and thus Fn.
    \item Prove |ψ|≥ sizef ψ ≥ sizef (\/ |Ts| i=0 Ts [i]) ≥ n·2n (using proposition 1 to show the last inequality).
  \end{enumerate}
\end{frame}

\begin{frame}{DNF to CNF}{Low-level}
  \begin{block}{Proposition (Blowup from $dual F_n$ to $BigAnd'$)}
   Let n > 0 be a natural number.
   Let Cs be a list of formulas.
   Let ψ= /\ |Cs| i=0 Cs[i].
   If all formulas in Cs are disjunctive and not tautological, and dual Fn is equivalent to ψ, then sizef ψ ≥ n·2n.
  \end{block}
  Proof:
  \begin{enumerate}
    \item Sizef (dual ψ) < n·2n.
    \item Dual ψ is equivalent to Fn.
    \item There exists a list Ts such that every formula contained in Ts is both conjunctive and satisfiable and such that dual ψ is syntactically equal to \/ |Ts| i=0Ts[i].
    \item Sizef (dual ψ) ≥ n·2n  (using steps 2, 3 and proposition 1).
  \end{enumerate}
  => The result of step 1 contradicts the result of step 4.
\end{frame}

\begin{frame}{DNF to CNF}{High-level}
  \begin{theorem}[Blowup from DNF to CNF]
    Let n be a natural number.
    There exists a formula ϕ in DNF with size |ϕ| = 10·n+1 such that every equivalent formula ψ in CNF has size |ψ| ≥ n·2n.
  \end{theorem}
  Proof:
    Analogous to the proof of theorem 1.
\end{frame}

\begin{frame}{Discussion}{}
  We defined new versions of BigOr and BigAnd from Michaelis and Nipkow, where we handle the base case differently to be able to prove |ψ| ≥ n· 2n (instead of |ψ| + 2 ≥ n · 2n).

  We introduced the additional size function sizef to be able to prove sizef (dual ϕ) = sizef ϕ (instead of |dual ϕ| ≤ 2 · |ϕ|), what we need to derive a contradiction in the proof of proposition 2.
\end{frame}

\begin{frame}{Conclusion}{}
  We formalized the exponential blowup of the formula size when transforming a formula into DNF or CNF in Isabelle/HOL.

  Our formalization may serve as a basis for further formalizations of quantitative results in propositional logic.

  Future work: Proof complexity lower bounds (e.g., for resolution proofs of the pidgeonhole principle)
